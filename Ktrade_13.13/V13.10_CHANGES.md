# KTrade — V13.10 (single-instance lock + Monte-Carlo robustness)

## 1. Single-instance lock on the agent (agent/ktrade_agent_v9.py)
The agent's `main()` now acquires a crash-safe `ProcessLock` for TRADING
invocations (the autonomous default and `--once`), exiting if another agent is
already running — so a stray manual launch alongside the systemd service can't
double-trade. Read-only modes (`--score-only`, `--risk-status`, `--crash-check`,
`--ask`, `--validate`) are exempt, so you can still scan/query while the loop runs.
Reuses the existing `ktrade_runtime/process_lock.py` (reclaims a stale lock from a
dead PID) and FAILS OPEN if that module is missing, so it can never block a
legitimate start. Lock file: `data/ktrade_agent.lock`.

## 2. Monte-Carlo robustness (agent/param_fragility.py)
`monte_carlo_analysis(trade_returns)` complements the v13.6 grid sweep: instead of
"is this parameter a lonely spike", it answers "is this Sharpe real or luck of a few
trades". Two tests, and a CORRECTION of the reviewer's version:
- The review said "shuffle trade order 1000x, check Sharpe > 0". That tests nothing —
  Sharpe (mean/std) is ORDER-INVARIANT, so shuffling can't change it.
- Correct instead: **(a) bootstrap** — resample trades WITH replacement, report
  P(Sharpe > 0) and the 5th-percentile bootstrap Sharpe (sampling robustness); and
  **(b) order-shuffle for DRAWDOWN**, which IS path-dependent — report the median and
  worst-5% max drawdown across orderings.
- Classification: ROBUST (P(Sharpe>0) >= 0.95 and 5th-pctile > 0), FRAGILE (< 0.90),
  else MIXED.

  python agent/param_fragility.py --mc-returns trades.csv    # one fractional return per line
  python agent/param_fragility.py --prices NVDA.csv --strategy macd   # (grid sweep, unchanged)

## Honest framing (unchanged)
The lock is safety; Monte-Carlo is a validation instrument. Neither creates an edge —
they help you keep from double-trading and help you judge whether an edge is real.
The Monte-Carlo test needs real per-trade returns (from paper trading or a backtest)
to say anything; run it once you've accumulated trades.
