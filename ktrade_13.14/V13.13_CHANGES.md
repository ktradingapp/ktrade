# KTrade — V13.13 (VIX feed wired, fail-safe)

Wires the real VIX/SPY feed into the agent — the one genuine pre-live gap flagged
across multiple reviews. The VIX ladder (risk-off, auto-hedge, 50% vol-sizing) and
the stale-VIX guard now use live data instead of the dormant 18.0 default.

## What changed
- **agent/broker_adapter.py** — `make_market_fn()` now fetches VIX via the data
  feed (Polygon `I:VIX`, yfinance `^VIX` fallback), STRICTLY: if no source succeeds
  it returns None and VIX is OMITTED, so the stale-VIX guard stays conservative
  (fail-safe) rather than trusting a calm default. Value is sanity-gated to 5–150 to
  reject bad ticks. SPY unchanged.
- **data/ktrade_data.py** — `get_vix(default=18.0)` gained a `default` param.
  `default=None` lets strict callers tell a real reading from a fallback. Back-compat
  preserved (existing callers still get 18.0).
- **agent/ktrade_agent_v9.py** — `main()` now passes `make_market_fn()` to the CEO
  (was `KTradeCEO()` with no feed). Gated by `KTRADE_WIRE_MARKET_FN=true` (default on).

## IMPORTANT caveat (verify on the VPS)
Polygon `I:VIX` needs an **index-data subscription**. If your Polygon plan doesn't
include it (and yfinance is unavailable/unreliable from the VPS datacenter IP), VIX
stays unwired — and the fail-safe means the guard keeps you conservative rather than
trading blind. So confirm on the VPS that `PolygonDataFeed().get_vix(default=None)`
returns a real number. This is verified-by-construction here (I can't hit Polygon
from the sandbox); the fail-safe/sanity logic is verified-by-execution.

## Verification
- `py_compile` clean. `test_v140_marketfn.py` **10/10**: real VIX flows through; a
  feed failure or an out-of-band tick omits VIX (fail-safe); SPY survives a VIX
  failure; strict `get_vix(default=None)` returns None and the 18.0 default is
  preserved. Full sweep: **25/25**. release-safety passes.

## Why this matters for a down market
On a high-volatility day the wired VIX makes the agent MORE conservative — risk-off
above VIX 30, half size above 25, close-all at 50. So the agent's designed response
to a downturn is to pull back, not pile in. Wiring VIX doesn't make it ready for live
money; it makes its crash protection actually function.
